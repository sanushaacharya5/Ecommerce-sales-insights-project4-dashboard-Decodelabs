# E-Commerce Sales Insights Dashboard

An interactive Tableau dashboard developed to analyze e-commerce sales performance, customer behavior, product demand, and marketing effectiveness.

## Tools Used

* Tableau Public
* Microsoft Excel

## Key Features

* Revenue Trend Analysis
* Product Performance Insights
* Marketing Channel Evaluation
* Payment Method Analysis
* Order Status Monitoring
* Interactive Filters and Visualizations

## Project Files

* Dashboard Screenshot (PNG)
* Tableau Public Dashboard Link
* Project Presentation (PDF)

## Key Insight

Instagram was the highest-performing marketing channel, while Chairs, Printers, and Laptops generated the highest revenue contributions.

## Conclusion

This dashboard provides a comprehensive view of e-commerce sales performance, customer behavior, and marketing effectiveness. 
Through interactive visualizations, it helps identify key revenue drivers, top-performing products, and customer preferences.
The insights generated can support data-driven decision-making and improve overall business performance.
